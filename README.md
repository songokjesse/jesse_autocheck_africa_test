## Description

[Nest](https://github.com/nestjs/nest) App that talks to HackerNews API.

## Installation
1. Clone the repository https://github.com/songokjesse/jesse_autocheck_africa_test.git to your local machine
2. After the cloning is completed enter the directory containing the code
3. Install the dependencies by running `npm install`


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
